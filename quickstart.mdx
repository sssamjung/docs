## 시작하기 전에

Pulse API를 사용하기 위해서는 다음이 필요합니다:

<Steps>
  <Step title="계정 발급">
    Pulse 네트워크 참여를 위해 [shs.block@shinhan.com](mailto:shs.block@shinhan.com)으로 연락하여 계정을 발급받으세요.
  </Step>
  <Step title="API 키 수령">
    계정 발급 후 Access Token을 수령합니다. 이 토큰은 모든 API 호출에 필요합니다.
  </Step>
  <Step title="환경 설정">
    개발 환경에 맞는 HTTP 클라이언트를 준비합니다.
  </Step>
</Steps>

## API 기본 정보

<ParamField body="Base URL" type="string">
  ```
  https://api.stopulse.co.kr/main/v1
  ```
</ParamField>

<ParamField body="Authentication" type="header">
  모든 API 요청에 `Authorization` 헤더가 필요합니다:
  ```
  Authorization: Bearer {your_access_token}
  ```
</ParamField>

## 첫 번째 API 호출

### 토큰 정보 조회

가장 간단한 API 호출로 토큰 정보를 조회해보겠습니다:

<CodeGroup>

```bash cURL
curl -X GET "https://api.stopulse.co.kr/main/v1/tokens/{token}/info" \
  -H "Authorization: Bearer {your_access_token}" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch(
  'https://api.stopulse.co.kr/main/v1/tokens/{token}/info',
  {
    method: 'GET',
    headers: {
      'Authorization': 'Bearer {your_access_token}',
      'Content-Type': 'application/json'
    }
  }
);

const data = await response.json();
console.log(data);
```

```python Python
import requests

url = "https://api.stopulse.co.kr/main/v1/tokens/{token}/info"
headers = {
    "Authorization": "Bearer {your_access_token}",
    "Content-Type": "application/json"
}

response = requests.get(url, headers=headers)
print(response.json())
```

```java Java
import java.net.http.*;
import java.net.URI;

HttpClient client = HttpClient.newHttpClient();
HttpRequest request = HttpRequest.newBuilder()
    .uri(URI.create("https://api.stopulse.co.kr/main/v1/tokens/{token}/info"))
    .header("Authorization", "Bearer {your_access_token}")
    .header("Content-Type", "application/json")
    .GET()
    .build();

HttpResponse<String> response = client.send(request,
    HttpResponse.BodyHandlers.ofString());
System.out.println(response.body());
```

</CodeGroup>

## 지갑 잔액 조회

투자자의 토큰 잔액을 조회합니다:

<CodeGroup>

```bash cURL
curl -X GET "https://api.stopulse.co.kr/main/v1/holders/{holder}/tokens/{token}/balance" \
  -H "Authorization: Bearer {your_access_token}"
```

```javascript JavaScript
const response = await fetch(
  'https://api.stopulse.co.kr/main/v1/holders/{holder}/tokens/{token}/balance',
  {
    headers: {
      'Authorization': 'Bearer {your_access_token}'
    }
  }
);

const balance = await response.json();
```

</CodeGroup>

### 응답 예시

```json
{
  "holder": "0xd14f723e859ec4f3edee57bf78ceb7c81de08dfb",
  "token": "PULSE-ST-001",
  "balance": "100"
}
```

## 에러 처리

API 호출 시 발생할 수 있는 일반적인 에러:

| HTTP Status | 설명 | 해결 방법 |
|-------------|------|----------|
| 401 | 인증 실패 | Access Token 확인 |
| 403 | 권한 없음 | API 권한 확인 |
| 404 | 리소스 없음 | 요청 경로 확인 |
| 500 | 서버 오류 | 지원팀 문의 |

<Warning>
  프로덕션 환경에서는 반드시 에러 핸들링을 구현하세요.
</Warning>

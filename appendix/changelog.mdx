## 2025년 1월

### v1.2.0 (2025-01-15)

<AccordionGroup>
  <Accordion title="새로운 기능">
    - **Partition API 개선**: 파티션별 잔액 조회 기능 추가

  </Accordion>
  <Accordion title="버그 수정">
    - 특정 조건에서 KYC 상태 조회 실패 문제 수정
    - Swap API의 동시성 이슈 해결
  </Accordion>
</AccordionGroup>

## 버전 관리 정책

### Semantic Versioning

Pulse API는 [Semantic Versioning](https://semver.org/) 정책을 따릅니다:

| 버전 | 설명 |
|------|------|
| **Major (X.0.0)** | 하위 호환성이 없는 변경 |
| **Minor (0.X.0)** | 하위 호환성 있는 새 기능 |
| **Patch (0.0.X)** | 하위 호환성 있는 버그 수정 |

### 사전 공지

<Warning>
  하위 호환성이 없는 변경(Breaking Change)은 최소 30일 전에 공지됩니다.
</Warning>

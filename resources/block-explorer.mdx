# Block Explorer

Pulse 네트워크의 Block Explorer는 **Blockscout** 기반으로 구축되어 있습니다. 블록, 트랜잭션, 주소, 스마트 컨트랙트를 조회할 수 있습니다.

## 주요 기능

### 블록 조회

최신 블록 및 블록 상세 정보를 조회합니다:

```
블록 상세 정보
├── Block Height (블록 번호)
├── Timestamp (생성 시각)
├── Transactions (포함된 트랜잭션 수)
├── Miner/Validator (블록 생성자)
├── Block Hash (블록 해시)
├── Parent Hash (이전 블록 해시)
├── Gas Used (사용된 Gas)
└── Size (블록 크기)
```

### 트랜잭션 조회

트랜잭션 해시로 상세 정보를 조회합니다:

```
트랜잭션 상세 정보
├── Transaction Hash (트랜잭션 해시)
├── Status (성공/실패)
├── Block (포함된 블록 번호)
├── Timestamp (처리 시각)
├── From (송신자 주소)
├── To (수신자 주소)
├── Value (전송 금액)
├── Gas Used (사용된 Gas)
├── Input Data (입력 데이터)
└── Logs (이벤트 로그)
```

### 주소 조회

지갑 주소의 잔액 및 거래 이력을 조회합니다:

```
주소 상세 정보
├── Address (지갑 주소)
├── Balance (잔액)
├── Transactions (거래 이력)
├── Token Transfers (토큰 전송 이력)
├── Internal Transactions (내부 트랜잭션)
└── Tokens (보유 토큰 목록)
```

### 토큰 조회

ERC-1400 토큰 정보를 조회합니다:

```
토큰 상세 정보
├── Token Name (토큰 이름)
├── Symbol (심볼)
├── Total Supply (총 발행량)
├── Holders (보유자 수)
├── Transfers (전송 이력)
└── Contract (컨트랙트 주소)
```

## 사용 방법

### 트랜잭션 검색

1. Block Explorer 접속
2. 검색창에 트랜잭션 해시 입력
3. 트랜잭션 상세 페이지에서 정보 확인

```
검색 예시:
0xabc123def456789...
```

### 주소 검색

1. 검색창에 지갑 주소 입력
2. 주소 상세 페이지에서 잔액, 거래 이력 확인

```
검색 예시:
0xd14f723e859ec4f3edee57bf78ceb7c81de08dfb
```

### 블록 검색

1. 검색창에 블록 번호 또는 블록 해시 입력
2. 블록 상세 페이지에서 정보 확인

```
검색 예시:
12345678 (블록 번호)
또는
0xblock123... (블록 해시)
```

## API 연동 URL 생성

Block Explorer URL을 프로그래밍 방식으로 생성할 수 있습니다:

```javascript
// 트랜잭션 상세 페이지 URL
const txUrl = `https://explorer.stopulse.co.kr/tx/${transactionHash}`;

// 주소 상세 페이지 URL
const addressUrl = `https://explorer.stopulse.co.kr/address/${walletAddress}`;

// 블록 상세 페이지 URL
const blockUrl = `https://explorer.stopulse.co.kr/block/${blockNumber}`;

// 토큰 상세 페이지 URL
const tokenUrl = `https://explorer.stopulse.co.kr/token/${tokenContractAddress}`;
```

### 활용 예시

```javascript
// 트랜잭션 완료 후 Explorer 링크 제공
async function transferAndGetExplorerLink(from, to, value) {
  const result = await pulseAPI.transfer({
    token: 'PULSE-ST-001',
    from,
    to,
    value
  });

  const explorerUrl = `https://explorer.stopulse.co.kr/tx/${result.transactionHash}`;

  return {
    transactionHash: result.transactionHash,
    explorerUrl
  };
}
```

## 스마트 컨트랙트 조회

### 컨트랙트 정보

```
컨트랙트 상세 정보
├── Contract Address (컨트랙트 주소)
├── Creator (생성자 주소)
├── Creation Transaction (생성 트랜잭션)
├── Contract Code (소스코드/바이트코드)
├── Read Contract (읽기 함수 실행)
└── Write Contract (쓰기 함수 실행)
```

### Verified Contract

소스코드가 검증된 컨트랙트는 다음 정보를 확인할 수 있습니다:

- Solidity 소스코드
- 컴파일러 버전
- ABI (Application Binary Interface)
- 생성자 인자

## 이벤트 로그 분석

### Transfer 이벤트

토큰 전송 시 발생하는 이벤트:

```
Event: Transfer
Topics:
  [0] 0xddf252ad... (event signature)
  [1] 0x000...from (indexed from address)
  [2] 0x000...to   (indexed to address)
Data:
  value: 100
```

### TransferByPartition 이벤트

파티션 토큰 전송 시 발생하는 이벤트:

```
Event: TransferByPartition
Topics:
  [0] 0x123abc... (event signature)
  [1] 0x000...partition (indexed partition)
  [2] 0x000...from (indexed from)
  [3] 0x000...to (indexed to)
Data:
  value: 100
  data: 0x...
```

## 모니터링 활용

### 주소 모니터링

특정 주소의 활동을 모니터링:

```javascript
// 주기적으로 주소 트랜잭션 확인
async function monitorAddress(address) {
  const transactions = await fetchAddressTransactions(address);

  for (const tx of transactions) {
    if (isNewTransaction(tx)) {
      console.log(`새 트랜잭션 감지: ${tx.hash}`);
      await processNewTransaction(tx);
    }
  }
}
```

### 토큰 모니터링

토큰 전송 이벤트 모니터링:

```javascript
// 토큰 Transfer 이벤트 모니터링
async function monitorTokenTransfers(tokenAddress) {
  const transfers = await fetchTokenTransfers(tokenAddress);

  for (const transfer of transfers) {
    console.log(`전송: ${transfer.from} → ${transfer.to}, ${transfer.value}`);
  }
}
```

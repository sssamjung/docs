# 인증 (Authentication)

Pulse API는 **Bearer Token** 방식의 인증을 사용합니다. 모든 API 요청에 Access Token을 포함해야 합니다.

## 인증 방식

### Authorization 헤더

모든 API 요청의 헤더에 Access Token을 포함합니다:

```
Authorization: Bearer {your_access_token}
```

### 요청 예시

<CodeGroup>

```bash cURL
curl -X GET "https://api.stopulse.co.kr/main/v1/tokens/{token}/info" \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..." \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch('https://api.stopulse.co.kr/main/v1/tokens/{token}/info', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...',
    'Content-Type': 'application/json'
  }
});
```

```python Python
import requests

headers = {
    'Authorization': 'Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...',
    'Content-Type': 'application/json'
}

response = requests.get(
    'https://api.stopulse.co.kr/main/v1/tokens/{token}/info',
    headers=headers
)
```

```java Java
HttpClient client = HttpClient.newHttpClient();
HttpRequest request = HttpRequest.newBuilder()
    .uri(URI.create("https://api.stopulse.co.kr/main/v1/tokens/{token}/info"))
    .header("Authorization", "Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...")
    .header("Content-Type", "application/json")
    .GET()
    .build();
```

</CodeGroup>

## 인증 에러

### 에러 코드

| HTTP Status | 에러 코드 | 설명 |
|-------------|----------|------|
| 401 | `UNAUTHORIZED` | Access Token 누락 또는 유효하지 않음 |
| 401 | `TOKEN_EXPIRED` | Access Token 만료 |
| 403 | `FORBIDDEN` | 해당 리소스에 대한 권한 없음 |

### 에러 응답 예시

```json
{
  "error": {
    "code": "UNAUTHORIZED",
    "message": "Invalid or missing access token",
    "details": "Authorization header is required"
  }
}
```

## Rate Limiting

API 호출에는 Rate Limit이 적용됩니다:

| 엔드포인트 유형 | 제한 |
|---------------|------|
| Read (조회) | 100 req/min |
| Write (트랜잭션) | 30 req/min |

Rate Limit 초과 시 `429 Too Many Requests` 응답이 반환됩니다.

```json
{
  "error": {
    "code": "RATE_LIMIT_EXCEEDED",
    "message": "Too many requests",
    "retryAfter": 60
  }
}
```
